# 《辟谣防卫战：相亲相爱一家人》作品介绍

## 一、设计理念

"柔性辟谣"——用游戏代替说教。

家庭群中的谣言传播，本质是代际沟通的断裂。传统辟谣往往居高临下、加剧对立。本设计以 MBTI 人格类型为框架，通过游戏化机制，让年轻人在模拟家庭群的场景中理解不同长辈的认知偏好，在"玩"中完成辟谣教育，降低防御心理，实现"代际共情"而非"代际说教"。

核心主张：打败谣言不仅需要科学数据，更需要沟通的温度与技巧。

## 二、核心设计

**人格系统**：6 种代表性人格（固执大爷、焦虑老妈、养生专家、阴谋论者、佛系邻居、爱孙奶奶），每种人格有不同辟谣抗性和卡牌效果。

**卡牌机制**：分为论据牌（理性攻击）、情绪牌（情感共鸣）、特殊牌（趣味转移），每回合 6 选 1。其中包含错误选项，玩家选择后触发科普弹窗，即时纠偏、建立知识迁移。

**数值系统**：家庭和谐度（HP）vs 谣言污染度，双轨并行，兼顾策略博弈与情感温度。

**场景还原**：模拟微信群聊界面，唤醒真实使用场景，增强代入感。

## 三、技术实现

- React 18 + Vite（前端框架与构建工具）
- Tailwind CSS v4（响应式样式适配）
- Framer Motion（卡牌动画与交互动效）
- Web Audio API（8-bit 像素风音效，无依赖原生实现）
- Capacitor（跨平台 APK 打包，实现本地独立运行）
- 纯前端架构，无后端依赖，所有状态存于内存

## 四、设计过程

1. 问题发现：调研家庭群谣言传播现象，发现"直接辟谣引发家庭矛盾"的痛点
2. 理论切入：引入 MBTI 人格类型，理解不同长辈的信息接受偏好
3. 机制设计：游戏化机制降低防御心理，错误选项即时科普，建立知识迁移
4. 迭代打磨：卡片动效、8-bit 音效、错误弹窗文案持续优化，平衡趣味性与教育性
5. 视觉定稿：像素风 UI + 微信群聊界面，唤起真实使用场景
